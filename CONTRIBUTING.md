# Contribuir a Presupuesto Digno

Gracias por querer contribuir. Este proyecto busca mantenerse simple, claro y útil para educación financiera básica.

## Principios

- Priorizar cambios pequeños y fáciles de revisar.
- Escribir textos claros en español.
- No agregar autenticación, bases de datos, APIs, IA ni arquitectura compleja para el MVP.
- No incluir métricas de impacto, usuarios o resultados que no estén verificados.

## Cómo proponer cambios

1. Abrí un issue o comentá la mejora que querés hacer.
2. Hacé un fork del repositorio.
3. Creá una rama descriptiva.
4. Ejecutá los tests con `pytest`.
5. Enviá un pull request con una descripción breve.

## Estilo de código

- Usar Python 3.12.
- Mantener type hints en la lógica principal.
- Mantener funciones simples y con docstrings.
- Agregar o actualizar tests cuando cambie la lógica de cálculo o recomendaciones.
